# Contributing

Thanks for helping improve StellarOps Board. This project is built for maintainers and contributors who care about clear, completable open-source work.

## Development Setup

```bash
npm install
npm run dev
```

Run checks before opening a pull request:

```bash
npm run lint
npm run typecheck
npm run test
npm run build
```

## Good Contributions

- Improve issue readiness scoring edge cases.
- Add test coverage for dashboard states.
- Improve documentation with concrete maintainer examples.
- Prepare GitHub API integration behind typed service boundaries.
- Add accessibility checks and keyboard interaction tests.

## Pull Request Guidance

Keep changes focused, explain the maintainer workflow impact, and avoid claiming official affiliation with Stellar, Soroban, Drips Wave, or GrantFox.
