# Maintainer Dashboard

The dashboard gives maintainers a single review surface for contributor readiness.

## Summary Metrics

The top summary cards show average issue readiness, ready issue count, repository health score, and label coverage. These metrics help maintainers decide whether a backlog is ready for a contributor program.

## Issue Table

The issue table displays title, repository, difficulty, labels, readiness score, status, assignee placeholder, and last updated date. Filters help maintainers find beginner work, label gaps, or weak issues.

## Health Cards

Repository health cards track README, CONTRIBUTING, CODE_OF_CONDUCT, LICENSE, docs, tests, CI workflow, issue templates, open contributor issues, and labels.

## Workflow Status

Workflow status shows whether issues are drafted, reviewed, labeled, ready for contributors, supported by PR review checklist, and backed by passing CI.
